Unsupported Blocks plugin is used to display an alert message whenever plugin is not supported.
It is designed for the mobile version of the editor

<div className="card">

    content={content}
    plugins={[pluginUnsupportedBlocks()]}
  />
</div>
<br />

## Usage

```jsx
/* Editor */
import { pluginUnsupportedBlocks } from 'wix-rich-content-plugin-unsupported-blocks';
import 'ricos-editor/dist/styles.min.css';
import 'wix-rich-content-plugin-unsupported-blocks/dist/styles.min.css';
import { RicosEditor } from 'ricos-editor';
```
