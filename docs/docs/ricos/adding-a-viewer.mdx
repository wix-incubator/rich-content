Now that you have an editor, let's add a viewer to display the content it creates.

```bash
npm i ricos-viewer
```

```jsx
import { RicosViewer } from 'ricos-viewer';
import 'ricos-viewer/dist/styles.min.css';

const content = {
  blocks: [
    {
      key: 'd79aa',
      text: "Hi there! I'm Ricos, your rich content framework. ðŸ‘¾",
      type: 'header-two',
      depth: 0,
      inlineStyleRanges: [],
      entityRanges: [],
      data: {},
    },
  ],
  entityMap: {},
  VERSION: '7.5.0',
};

<RicosViewer content={content} />;
```

<div className="card">
  <RicosViewer content={content} />
</div>

Now let's add some plugins to Ricos and display the new content.

```bash
npm i wix-rich-content-plugin-video wix-rich-content-plugin-divider
```

```jsx
import { RicosViewer } from 'ricos-viewer';
import 'ricos-viewer/dist/styles.min.css';

import { pluginVideo } from 'wix-rich-content-plugin-video/dist/module.viewer.cjs';
import 'wix-rich-content-plugin-video/dist/styles.min.css';

import { pluginDivider } from 'wix-rich-content-plugin-divider/dist/module.viewer.cjs';
import 'wix-rich-content-plugin-divider/dist/styles.min.css';

<RicosViewer content={contentWithPlugins} plugins={[pluginDivider(), pluginVideo()]} />;
```

<div className="card">
  <RicosViewer content={contentWithPlugins} plugins={[pluginDivider(), pluginVideo()]} />
</div>
