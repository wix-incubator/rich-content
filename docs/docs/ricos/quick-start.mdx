Start from installing `WixRichContent` and it's open-source counterpart.

```bash
npm i @wix/rich-content wix-rich-content-wrapper
```

Then add a basic rich text editor to your app. Try typing, and add some styling to your text.

```jsx
import { WixRichContent } from "@wix/rich-content";
import { RichContentEditor } from "wix-rich-content-editor";

<WixRichContent consumer={"your app name for BI"} isEditor>
	<RichContentEditor placeholder={"Type here!"} />
</WixRichContent>;
```

</div>

Now let's add some plugins. This is how you can add videos and dividers.

```bash
npm i wix-rich-content-plugin-video wix-rich-content-plugin-divider
```

```jsx
import { WixRichContent } from "@wix/rich-content";
import { RichContentEditor } from "wix-rich-content-editor";

import { pluginVideo } from "wix-rich-content-plugin-video";
import "wix-rich-content-plugin-video/dist/styles.min.css";

import { pluginDivider } from "wix-rich-content-plugin-divider";
import "wix-rich-content-plugin-divider/dist/styles.min.css";

<WixRichContent
	consumer={"your app name for BI"}
	isEditor
	plugins={[pluginDivider(), pluginVideo()]}
>
	<RichContentEditor placeholder={"Type here!"} />
</WixRichContent>;
```

<div class="docsEditor">

There you go! A rich content editor with plugins. Now go ahead and add some more plugins!
