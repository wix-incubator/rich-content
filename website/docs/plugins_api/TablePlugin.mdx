## Usage

```jsx
/* Editor */
import { RicosEditor } from 'ricos-editor';
import { pluginTable } from 'wix-rich-content-plugin-table';
import 'ricos-editor/dist/styles.min.css';
import 'wix-rich-content-plugin-table/dist/styles.min.css';
<RicosEditor plugins={[pluginTable()]} />;

/* Viewer */
import { RicosViewer } from 'ricos-viewer';
import { pluginTable } from 'wix-rich-content-plugin-table/dist/module.viewer';
import 'wix-rich-content-plugin-table/dist/styles.min.css';
<RicosViewer plugins={[pluginTable()]} />;
```

## Table Config API

### `innerRCEPlugins`

Controls which plugins will be available to add to the table cells.
If innerRCEPlugins is not defined - the available plugins will be those you already consume out of the SupportedTablePlugins

```jsx
  innerRCEPlugins?: SupportedTablePlugins[];
```

Example value:

```jsx
[createImagePlugin, createDividerPlugin, createVideoPlugin];
```
